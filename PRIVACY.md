# CanvasUtils Privacy Policy

Last updated: 2026-03-14

Overview
CanvasUtils is a simple browser extension that toggles line-wrapping on the active tab to improve readability. It does not collect or transmit personal data.

Data collection
- Personally identifiable information: None
- Health, financial, authentication info: None
- Personal communications, location, web history: None
- User activity (keystrokes, network monitoring): None
- Website content: The extension reads and modifies page content locally to apply or revert wrapping. This content is not collected, stored, or sent to external servers.

Permissions and why they are used
- activeTab: Required to run the content script on the page the user selects so the extension can toggle wrap/revert locally.
- storage (optional): If enabled, used only to store a simple local preference (e.g., last toggle state). No personal data is stored or uploaded.
- Host permissions: None requested.
- Remote code: None. All code is bundled in the extension package.

Data sharing and third parties
- CanvasUtils does not sell, share, or transfer user data to third parties.
- No analytics, advertising, or external telemetry services are used.

Security
All processing is performed locally in the user's browser. No external servers are contacted by the extension.
